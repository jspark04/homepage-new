Flonomix Website Redesign Concept

Languages: HTML5, CSS3, Javascript, PHP

Description: I found out about Bootstrap from a co-worker and wanted to learn how to use it effectively to quickly produce decent looking sites or site mockups. In quasi-layman's terms, Bootstrap is essentially a package with pre-made CSS and a few Javascript plugins meant to be used to kick-start development of a website without having to start from scratch. Much of this Flonomix website is straight Bootstrap. There's a bit of jQuery used to do some form validation, an AJAX call to a PHP script, and I wrote a small PHP script to be able to mail the form contents and respond with an HTML page if AJAX/Javascript isn't supported by the browser.
